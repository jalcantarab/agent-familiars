# Changelog

## v0.1.0 - Initial Public Release

- Ships 66 installable Codex pet bundles.
- Adds Signal Surface as a non-creature utility familiar for glanceable state
  changes.
- Adds the `state-instruments` pack: Signal Surface, Intent Compass, Thread
  Totem, Pocket Oracle, Loop Loom, and Tide Stone.
- Adds a chooser guide for picking a first familiar by screen mood, job fit,
  and personality.
- Adds automated pet routine guidance plus a local variant rotation utility for
  time-of-day or lab use.
- Includes the complete first-50 workflow set through `--pack first-50`.
- Includes the product-folklore pack through `--pack product-tropes`.
- Adds per-pet README pages with all nine animation states.
- Adds generated preview GIFs, contact sheets, and social showcase reels.
- Adds a generated README banner built from committed spritesheets and moves the
  product-folklore catalog ahead of the first-50 catalog.
- Embeds the main showcase reel near the top of the README and adds compact
  animated chooser cards for product judgment, developer workflow, and quiet
  instruments.
- Adds a README surface compatibility FAQ for Codex, Claude, MCP clients, and
  exported media surfaces.
- Adds reusable sequence recipes for local GIF, MP4, poster, and pack-comparison
  exports.
- Adds local and raw-GitHub install flows through `scripts/install_pet.py`.
- Adds the installable `familiars` CLI and local MCP server through
  `familiars-mcp`.
- Adds an MCP client smoke test that launches the real stdio server, calls
  resources/tools/prompts, renders a poster, installs one familiar, and checks
  output-root safety.
- Adds validation for pet bundles, animation design specs, README media budget,
  README cards, previews, reels, and installer smoke tests.
- Adds a release-readiness check for public files, issue intake, catalog packs,
  and tracked-file hygiene.
- Documents licensing, attribution, contribution rules, security reporting, and
  release workflow.
