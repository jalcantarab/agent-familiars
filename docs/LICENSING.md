# Licensing

Familiars is meant to be easy to install, remix, and share without hiding where
any asset came from.

## Default License

The root [LICENSE](../LICENSE) is MIT. Unless a file, pet README, catalog entry,
or notice says otherwise, MIT covers:

- repository scripts and validation/rendering tools
- original documentation and catalog metadata
- original Familiars pet bundles in `pets/`
- generated preview GIFs, contact sheets, and showcase media created from
  original Familiars pets

This keeps reuse low-friction for developers. MIT requires preserving copyright
and license notices, and it provides the work without warranty.

Beyond required license notices, attribution is appreciated: if Familiars helps
your project, a link back to this repository is welcome. This is not an extra
condition on the MIT-licensed original assets.

## Per-Pet Licensing

Each pet can have its own provenance. Check these files before redistributing a
pet outside this repository:

- [catalog/pets.json](../catalog/pets.json)
- the pet's own `README.md`
- [NOTICE.md](../NOTICE.md)
- any third-party license file under [licenses/](../licenses)

Original pets by Jorge Alcantara / Zentrik are MIT for the bundle distributed in
this repository. Imported pets keep their upstream license. The root MIT license
does not replace upstream attribution requirements for imported assets.

## Brand Marks

The Zentrik name, logo, and related brand assets appear here for project
identity, attribution, examples, and the Zentri reference pet. The MIT license
does not grant standalone trademark or brand rights, and reuse should not imply
Zentrik endorsement.

## Adding Pets

Original contributions should use MIT unless maintainers agree to another
license before merge. Imported pets must include a source URL, creator or author
name, license name, and a copied license text when the license is not already
covered by the root MIT license.

Do not add:

- company logos, product marks, mascots, slogans, or branded UI you do not own
- fan art or characters from games, shows, comics, or films
- assets under non-commercial, no-derivatives, or unclear licenses
- assets that require attribution we cannot preserve clearly

The goal is generous reuse with boring provenance.
