## Install

```bash
mkdir -p ~/.codex/pets
cp -R pets/backlog-archaeologist ~/.codex/pets/
```

Then refresh custom pets in Codex and select `Backlog Archaeologist`.

## Motion Notes

- `waiting`: holds up an abstract fossil slab and asks whether the old thing still matters.
- `running`: scrapes an attached sediment layer to reveal one cleaner artifact shape.
- `review`: polishes the fossil shell until the evidence is ready to judge.
- `failed`: gets covered by attached fossil strata while goggles peek through.
