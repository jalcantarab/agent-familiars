## Install

```bash
mkdir -p ~/.codex/pets
cp -R pets/security-sentinel ~/.codex/pets/
```

Then refresh custom pets in Codex and select `Security Sentinel`.

## Motion Notes

- `idle`: holds a guarded shield-body stance with a narrow cyan visor.

- `running-right`: steps right through controlled boundary checks.

- `running-left`: steps left through controlled boundary checks.

- `waving`: opens one guard hand in a careful access acknowledgement.

- `jumping`: performs a controlled guard hop without losing defensive shape.

- `failed`: shield posture cracks inward and the head lowers.

- `waiting`: locks into checkpoint posture until the risk choice is made.

- `running`: pivots through threat-boundary angles while the visor scans.

- `review`: narrows into a precise inspection stance at the trust boundary.
