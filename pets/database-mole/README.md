## Install

```bash
mkdir -p ~/.codex/pets
cp -R pets/database-mole ~/.codex/pets/
```

Then refresh custom pets in Codex and select `Database Mole`.

## Motion Notes

- `idle`: holds the row blocks close while nose and paws breathe softly.
- `running-right` / `running-left`: shuffles nose-first like a burrow under the schema.
- `waving`: gives a small paw greeting without dropping the rows.
- `jumping`: pops up from burrow posture with paws tucked close.
- `failed`: backs into the burrow posture and covers its face.
- `waiting`: emerges nose-first and pauses for migration approval.
- `running`: packs attached row-like blocks into tidy layers.
- `review`: sniffs a schema layer and taps twice with one paw.
