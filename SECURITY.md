# Security Policy

Familiars is a public collection of static pet assets, small installer
scripts, and documentation.

## Reporting A Vulnerability

Please report security issues privately through GitHub private vulnerability
reporting.

If private vulnerability reporting is unavailable, open a minimal public issue
asking for a secure contact path. Do not include exploit details, suspected
secrets, or supply-chain specifics in that public issue.

## Scope

Security reports are most useful when they involve:

- unsafe installer behavior
- dependency or supply-chain risk
- accidentally committed secrets or private data
- misleading attribution that could create trust or licensing risk

Pet design requests, broken previews, and installation help belong in normal issues.
